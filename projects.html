Undergraduate Thesis

IoT-Based Balance and Tremor Exercise Board for Parkinson's Disease Patients

An IoT-connected exercise board that helps Parkinson's disease patients perform guided balance and tremor exercises, while continuously tracking blood pressure and syncing session data to a companion Android app built with Flutter.

Arduino / Microcontroller IoT Sensors Blood Pressure Module Flutter Android Studio

Background

Developed as my undergraduate thesis at Jose Rizal University, this project responds to the balance instability and hand/limb tremors common in Parkinson's disease, which make guided physical therapy exercises difficult to perform and monitor consistently at home.

Problem Statement

Patients often lack an affordable way to perform balance and tremor exercises with real-time feedback, and caregivers have no easy way to track exercise consistency or vital signs like blood pressure between clinic visits.

Solution

A physical exercise board fitted with balance and tremor-sensing hardware plus an integrated blood pressure monitor, all connected over IoT to a companion Android application that logs sessions, displays progress, and surfaces vitals in an easy-to-read format for the patient and caregiver.

Hardware

  • Balance/tremor exercise board with embedded sensors
  • Blood pressure monitoring module
  • Microcontroller for sensor data acquisition and wireless transmission

Software

  • Android companion app developed in Android Studio using Flutter/Dart
  • Real-time data sync between the exercise board and the mobile app
  • Session history and blood pressure trend display for patients and caregivers

Challenges

Calibrating tremor/balance sensors for reliable, consistent readings across different patients, and building a mobile interface simple enough for elderly users while still surfacing meaningful health data.

Cybernate — AI Scam Detection Application

A mobile application that uses AI to help everyday users identify potential scam messages, links, and calls before they fall victim — built as an Android app to make scam awareness accessible and portable.

Flutter Android Studio AI / Machine Learning Python

Background

Scams delivered through text messages, links, and calls are a widespread and growing risk, especially for users who aren't familiar with common red flags. Cybernate was built to give people a quick, AI-assisted second opinion before they engage with a suspicious message.

Problem Statement

Most people have no simple, on-the-go way to check whether a message or link they've received is likely to be a scam, and often only realize after damage is already done.

Solution

An Android application, developed with Flutter and Android Studio, that applies AI/ML techniques to flag suspicious content and give users a clear risk assessment, packaged in an approachable mobile interface.

Software

  • Flutter-based Android app UI/UX
  • AI/ML-driven scam and risk detection logic
  • Simple, recruiter/user-friendly result screens (safe / suspicious / high-risk)

Features

  • Scam-likelihood analysis for messages and links
  • Mobile-first design for on-the-go checks
  • Applies AI/ML fundamentals learned through coursework and self-directed practice

Challenges

Balancing detection accuracy with a simple, non-technical presentation so the app stays usable for people with no cybersecurity background.

AI Application

AI Drowsiness Detection Project

A real-time computer vision system that monitors a driver's or operator's face through a webcam, detects signs of drowsiness such as prolonged eye closure and yawning, and triggers an alert to help prevent fatigue-related accidents.

Python OpenCV Facial Landmark Detection Machine Learning

Background

Driver and operator fatigue is a major contributor to road accidents and workplace incidents, and drowsiness often builds up gradually in ways that are easy to miss without an external monitor. This project explores how computer vision and AI can provide that monitor in real time.

Problem Statement

There's no simple, low-cost way for a driver or machine operator to be automatically warned the moment their alertness starts to drop, before a lapse in attention leads to an accident.

Solution

A webcam-based monitoring system that tracks facial landmarks in real time, calculates indicators like eye-closure duration and yawn frequency, and sounds an alert as soon as the readings cross a drowsiness threshold.

How It Works

  • Captures live video from a webcam and detects the face and eye regions frame by frame
  • Tracks facial landmarks to measure eye-aspect-ratio and detect prolonged eye closure and yawning
  • Flags a drowsiness event when closure duration crosses a set threshold and triggers an audible/visual alert
